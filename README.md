# Ilhan Ahmed Ramadhan — Personal Resume Website

A multi-page personal resume website built using only HTML.
This is Capstone Project 1 from the **Udemy 100 Days of Code: 
Full Stack Web Development** course by Dr. Angela Yu.

## Pages
- **Home** — Resume with summary, education, work experience, skills, and awards
- **Hobbies** — Personal interests including football, gaming, and coding
- **Contact** — Contact details including email, phone, WhatsApp, LinkedIn, and GitHub

## Built With
- HTML5 only — no CSS, no JavaScript
